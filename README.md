# 🚀 Cyber Risk Assessment & Threat Intelligence Platform

## 📌 Project Overview
This project is a **Cyber Risk Assessment and Threat Intelligence Platform** developed to identify vulnerabilities, analyze risks, and provide actionable security insights. It integrates multiple tools and APIs to automate scanning, generate reports, and visualize security data through an interactive dashboard.

---

## 🎯 Objectives
- Detect vulnerabilities in systems and networks  
- Assign risk scores based on severity  
- Provide real-time threat intelligence  
- Generate detailed security reports  
- Visualize data using dashboards  

---

## ⚙️ Features

### 🔍 Vulnerability Scanning
- Uses **Nmap** for network scanning  
- Detects open ports and services  
- Identifies potential security risks  

### 📊 Interactive Dashboard
- Pie charts and graphs for analysis  
- Risk distribution visualization  
- Clean and user-friendly UI  

### 📧 Email Notification System
- Sends alerts after scan completion  
- Includes risk summary in email  

### 📄 Report Generation
- Downloadable reports (PDF/HTML)  
- Includes vulnerabilities and risk levels  

### 📈 Data Visualization
- Multiple charts for each module  
- Real-time insights for better decisions
